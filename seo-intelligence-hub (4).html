📌 How to export each report

GA4 Reports Snapshot

  1. GA4 → Reports → Overview
  2. Set date range (last 90 days)
  3. Click download icon → CSV

GSC Search Queries

  1. Search Console → Performance
  2. Set date range → Export → CSV
  3. Includes queries, clicks, impressions, CTR, position

GSC Pages Report

  1. Search Console → Performance
  2. Click "Pages" tab → Export → CSV

Keyword Research (Ubersuggest / SEMrush / Ahrefs)

  1. Run keyword analysis for client domain
  2. Export as CSV (any format works)
  3. Include: keyword, volume, difficulty, position
